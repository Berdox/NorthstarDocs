## File Location of Weapon Models
---
 The default location for the weapon model files is in the mp_common vpk.

   - ````englishclient_mp_common.bsp.pak000_dir.vpk\models\weapons\<Folder>` ```

 `<Folder>` is the corresponding value in the different tables. Change `<Folder>` to any value in the folder column for that weapon.

   - e.g. ``` `englishclient_mp_common.bsp.pak000_dir.vpk\models\weapons\rspn101` ```
